# EvoGenesis / EvolGenesis

Welcome to **EvolGenesis**, an open-source, evolving embedded systems and robotics platform.

This repository is designed to host the firmware, hardware layouts, web dashboard, and mobile apps that power the EvolGenesis ecosystem.

## Documentation

To learn more about the project, brand concepts, and repository architecture, check out the documentation:

* [Brand Concept & Architecture](file:///home/sohidul/developer_workspace/projects/aloevol/evoGenesis/docs/brand_concept.md)

## Repository Structure Overview

* **`firmware/`**: Rust-based firmware.
* **`web-dashboard/`**: Telemetry and control dashboard.
* **`mobile-app/`**: Companion mobile application.
* **`hardware/`**: Schematics, PCB designs, and wiring diagrams.
* **`simulations/`**: Simulation environments.
* **`experiments/`**: Prototyping sandbox.
* **`docs/`**: Documentation and learning guides.
